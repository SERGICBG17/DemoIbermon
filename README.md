## Descripción

**Ibermon** es un videojuego de rol inspirado en Pokémon, ambientado en una aventura única con criaturas ibéricas. Explora un vasto mundo, captura Ibermons, combate con otros entrenadores y descubre los secretos que esconde este universo.

Esta versión **Demo** te permite experimentar las mecánicas básicas del juego y probar una porción del contenido que estará disponible en la versión completa.

### Características de la Demo

-  **Objetivo principal**: Captura 5 Ibermons para completar la demo
-  **Exploración libre**: Recorre diferentes zonas del mapa
-  **Interacción con NPCs**: Habla con personajes para obtener información y misiones
-  **Sistema de objetos**: Interactúa con objetos del mundo
-  **Combates**: Sistema de batallas por turnos
-  **Sistema de captura**: Atrapa Ibermons salvajes

## Objetivo de la Demo

Para completar la demo exitosamente, deberás:
1. Capturar **5 Ibermons** diferentes
2. Formar tu equipo completo
3. Explorar el mundo disponible
4. Interactuar con NPCs y objetos

## Instalación

### Windows

1. **Descarga** el juego desde la sección [Tags](https://github.com/SERGICBG17/DemoIbermon/releases/tag/Windows) o navega a la carpeta `Ejecutables/Windows`

2. **Extrae** el archivo comprimido en la ubicación deseada

3. **Ejecuta** el archivo `Ibermon.exe`

4. ¡Empieza a jugar!

> **Nota:** Si Windows Defender bloquea la ejecución, haz clic en "Más información" y luego en "Ejecutar de todas formas".

### Android

1. **Descarga** el archivo `.apk` desde la carpeta `Ejecutables/Android` o desde [Tags](https://github.com/SERGICBG17/DemoIbermon/releases/tag/Android)

2. **Habilita** la instalación de aplicaciones de orígenes desconocidos:
   - Ve a **Ajustes** → **Seguridad**
   - Activa **Orígenes desconocidos** o **Instalar aplicaciones desconocidas**

3. **Abre** el archivo `.apk` descargado

4. **Instala** la aplicación siguiendo las instrucciones en pantalla

5. ¡Abre la app y comienza tu aventura!

> **Importante:** La primera vez que inicies el juego puede tardar un poco en cargar.
